> **8 autonomous AI agents** collaborating to produce institutional-quality research reports in **2 hours** (vs 2 weeks manual). Generated **1,240 reports** for investment firms and strategy teams.

## 👥 The Research Crew
| Agent | Role | Tools |
|-------|------|-------|
| **Research Director** | Orchestrates crew, sets research agenda | Planning, delegation |
| **Market Analyst** | Market sizing, trends, forecasts | Web search, financial APIs |
| **Competitor Intel** | Competitive landscape mapping | Web scraping, LinkedIn |
| **Financial Modeler** | Revenue projections, unit economics | Python, spreadsheets |
| **Sentiment Analyst** | News, social, investor sentiment | NLP, Twitter API |
| **Risk Assessor** | Regulatory, market, tech risks | Legal databases |
| **Data Scientist** | Statistical analysis, modeling | Python, BigQuery |
| **Report Writer** | Synthesizes into executive report | Writing, formatting |
